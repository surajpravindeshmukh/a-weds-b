# 🌸 मराठी विवाह · सहचर्य प्रेम 🌸

एक सुंदर, हलकं आणि भावनिक **Marathi Digital Wedding Invitation Website**.  
ही वेबसाईट पारंपरिक मराठी विवाहाची भावना आधुनिक वेब तंत्रज्ञानासोबत सादर करते.

---

## ✨ Features

- 💍 **Dynamic Bride & Groom Names**
- 📅 **Dynamic Date, Time & Venue**
- 🪔 **Marathi Vow Words Background Animation**
- 🎵 **Auto-playing Wedding Background Music**
- 📍 **Clickable Venue (Google Maps Integration)**
- 📜 **Clickable पत्रिका Download (Image)**
- 📱 **Mobile-friendly & Responsive Design**

## ⚙️ Configuration

You can easily update wedding details from the JavaScript section:

```js
let brideName = "स्नेहल";
let groomName = "रोहन";
let weddingDate = "शनिवार, १५ मार्च २०२६";
let weddingTime = "दुपारी १ वा ५८ मि.";
let weddingLocation = "सूर्य सांस्कृतिक भवन, गडहिंग्लज";
let rsvpEmail = "example@email.com";
```

All changes reflect instantly on the UI.
